现代社会学把"社会"定义为:"生活在一个特定地理区域内、彼此互动、并共享某种共同文化的一群人。"

而我们的定义,则是从"形而上学优先"的角度来看待社会的。我们把社会定义为一种形而上学的有机体或"心智",它由许多单独的心智构成,这些心智或明或暗地同意成为该社会的一部分。

- 隐性的同意,是出于方便。
- 显性的同意,是出于自愿选择。

> 社会是一种由众多个体心智构成的形而上学有机体

## 超灵—个灵,梵我(Paramatman)—个体我(Atman)

我们的定义运用了超物理学中"超灵—个灵"(oversoul-soul)这一概念,而这一概念又源自"复合观念由简单观念构成"这一机制(也就是说,每一个灵魂,都是至上实体的一个观念)。

这一点,巴门尼德以及东方哲学都曾对此作出解释。

{{< q a="巴门尼德" c="柏拉图《巴门尼德篇》,第二章" >}}
"太一"本身,一旦因"存在"而被分解为各个部分,便成为众多且无限的存在。这意味着,不仅拥有存在的"太一"是众多的,"太一"本身,一旦因存在而被分割,也必然是众多的。
{{< /q >}}

{{< q a="P·R·萨卡尔" c="《观念与意识形态》,第一章" >}}
既然心智是存在的,那么"心之载体"也是存在的,梵我(Paramatman)的映射就必然保留下来。因此,阿特曼(Átman)始终与个体灵魂(jiivátman)相连。终极的知晓者、行为者以及存在的根基,都在于阿特曼之中。
{{< /q >}}

科学认为,抽象的心智源自物质的大脑。

而超社会学恰恰相反,认为大脑的活动,是源自抽象的心智。这正是为什么我们用"灵魂"或"以太心智"这样的词语来指代一个社会。

由此,"社会"这个词可以指:
- 社会之魂,或者说
  - 这是其文化在抽象层面上的源头(比如,为什么中国人选择用筷子——也就是说,为什么中国这个以太集体心智吸收并同化了"用筷子"这个观念,而不是选择用手,或者用勺子叉子这类竞争性的观念)
- 物质层面的社会
  - 这是其文化在物质层面上的源头(比如,现实中的中国人使用实实在在的筷子)

简而言之:
- 灵魂,作为至上实体的一部分,是正力与负力这两种力量(历史上称为湿婆与萨克蒂,亦即阳与阴)的产物
- 这两种力量彼此对抗,由此产生了"波"
- 因此,"波"是绝对实体失去稳定性所产生的效应,而这又导致了"存在"与"至上实体"的产生,二者都具有波动的本质
- 这种波动的本质,创造出了多样性与周期性循环

把一个社会称为"灵魂",这样做能让我们:
- 使它摆脱物质层面的限制
- 赋予它类似声音与音乐那样灵活多变的特质

这些灵活的特质,类似于波——波正是传递情感信息的媒介。举例来说,一段曲调可以逐渐地、也可以瞬间地从旋律性转变为节奏性,就好比一个人的心智,可以逐渐地、也可以瞬间地从想吃美国菜转变为想吃中国菜。

### 社会 与 伙伴关系 的区别

一个"社会",至少需要三个或三个以上、为共同目标或共同利益而努力的实体。

而"伙伴关系"则有所不同,它至少只需要两个实体。

因此,一对没有孩子的夫妻,只算是一种伙伴关系。而一旦有了孩子,他们就成为了一个"社会"。

家庭,就是社会最常见的一个例子。

一支篮球队是一个社会,这不同于网球双打选手所构成的伙伴关系。

既然一支球队是一个社会,那么它也同样会受到"社会周期"的支配,这种周期表现为连胜或连败的状态。这些周期,正代表着这个社会有机体欲望之流的变化。
{{< /box >}}

{{< q a="休谟" c="《人性论》第一卷,第三部分,第一节" >}}
所有人的心智,在情感与运作方式上都是相似的。没有人会产生一种别人完全无法体会的情感。就好比一排绷紧的琴弦,其中一根弦的振动会传递给其余的琴弦。同样,各种情感也很容易从一个人身上传递给另一个人,并在每一个人身上引发相应的反应。
{{< /q >}}

{{< q a="苏格拉底" c="《理想国》简明版,第十卷" >}}
很少有人会去反思:别人身上的邪恶,也会有一部分传染到自己身上。因此,那种因目睹他人不幸而不断积累的悲伤情绪,会变得越来越难以压抑。
{{< /q >}}

> 我们的这个定义,赋予了社会灵活多变、类似波动的特质

正是这种灵活性,使得灵魂之间能够通过直觉传递并接收那些看不见的信息。用我们前面音乐的比喻来说,乐团可以从指挥那里获得视觉上的暗示,从而加快节奏,或者让弦乐部分演奏得更响亮。

这种灵活的特质,也使得灵魂之间能够彼此融合、达成一致,这正体现在"心意相通"或"想法一致"这样的说法中。这就好比一个乐团,增添了一些与自身风格相契合的新乐器。

这也解释了,为什么中国人采纳了筷子,而欧洲人却采纳了勺子和叉子:

- "筷子"这个观念的波长,与中国这个社会有机体的波长相匹配,因此被同化吸收
- "勺叉"这个观念的波长,则与欧洲这个社会有机体的波长相匹配

只要了解"筷子"这个观念与"勺叉"这个观念之间波长的差异,我们就能找到一个中国人和欧洲人都能接受的中间振动频率。如此一来,双方社会的波长就能够彼此和谐、达成一致,而不是相互对立、彼此排斥。

与"波"不同的是,物质与实体的东西,无法在不失去各自独立身份的前提下,同时占据同一个物理"位置"。

这种**心智或心灵之波的融合**,表现为:
- 社会的形成
- 友谊的建立
- 公司的成立
- 国家的建立,等等

举例来说:
- 一个家庭的开始,是一个男人的灵魂同意与一个女人的灵魂结合,而后再加入孩子的灵魂。而离婚,则是这个家庭灵魂重新分解为各个组成灵魂的过程
- 一家公司的开始,是各位创办人的灵魂同意结合成一个商业组织
- 一支运动队的开始,是队员们的灵魂同意一起比赛


### 社会通过"共同利益"实现规模的扩展

"波"能够彼此融合、形成更大的波,这一点在管弦乐团中体现得最为明显——单个乐器的声音,可以融合成一个统一的、更大、更复杂的声波。

既然社会是由个体灵魂之波构成的,那么它们就能够彼此一致、相互融合,形成国家;国家又能进一步融合,形成邦联;而邦联,最终能够形成一个全球性的人类社会。

这种统一性,是建立在这个群体中各种波的相似性之上的。而在一个社会当中,这种相似性,就被称为"共同利益"。

社会超物理学的目标,正是通过阐明灵魂的种种特性,以及苏格拉底、大卫·休谟、印度教、佛教与道教所阐述的"共同利益"的运作机制,来促成这样一个统一的社会。
